# 🤝 Contributing

### Overview

CubePay repositories are primarily maintained by internal teams.

At this time, we do not actively accept external contributions.

### Feedback and Issues

If you encounter a bug or have suggestions:

- Open a GitHub issue (where enabled), or  
- Use official support channels where applicable  

### Notes

- Pull requests from external contributors may not be reviewed or accepted  
- For integration or platform-related requests, please refer to official CubePay channels
