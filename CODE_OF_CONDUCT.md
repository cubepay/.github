# 📘 Code of Conduct

### Our Commitment

CubePay is committed to fostering a professional, respectful, and inclusive environment for everyone interacting with our repositories and community.

### Expected Behavior

Participants are expected to:

- Be respectful and considerate in communication  
- Provide constructive feedback  
- Focus on collaboration and problem-solving  
- Use professional language at all times  

### Unacceptable Behavior

The following will not be tolerated:

- Harassment, discrimination, or abusive conduct  
- Personal attacks or inflammatory language  
- Disruptive or unprofessional behavior  

### Scope

This Code of Conduct applies to all interactions within CubePay GitHub repositories, including:

- Issues  
- Pull requests  
- Discussions (where enabled)  

### Reporting

If you experience or observe behavior that violates this Code of Conduct, please report it to:

- 📧 security@cubepayment.com  

All reports will be handled confidentially.

### Enforcement

CubePay reserves the right to take appropriate action, including:

- Removing content  
- Restricting access  
- Blocking users from participation
