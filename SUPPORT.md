# 💬 Support

### Overview

This repository is maintained by CubePay for integration and developer enablement.  
GitHub issues are **not used for general support requests**.

### Getting Help

For platform access, onboarding, or technical support, please use the official channels:

- 🌐 https://www.radiumone.io  
- 🛠 https://support.radiumone.io  

These channels are monitored and will ensure your request is routed appropriately.

### Documentation

For integration guides and developer resources:

- 📘 https://docs.radiumone.io  

### Reporting Issues

- **Bugs or improvements specific to this repository** may be raised via GitHub Issues  
- Please provide clear steps to reproduce and relevant context  

For all other matters (account, transactions, credentials, production issues), use official support channels.

### Security Issues

If you believe you have found a security vulnerability, **do not open a public issue**.

Please refer to our security policy:

- 📄 See `SECURITY.md`

### Notes

- Support is provided through official CubePay channels only  
- Response times and SLAs are managed outside of GitHub
